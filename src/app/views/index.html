Exercise tracker

Create a New User

POST /api/users

User name should be at least 5 lowercase letters and/or numbers.

Add exercises

POST /api/users/:_id/exercises

Delete User

POST /api/users/:_id/delete

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
